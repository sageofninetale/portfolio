# UX-Focused 3D Portfolio

## What This Is

An immersive, interactive, and scroll-driven 3D WebGL portfolio for Aryan Subhash, a Data & Analytics Professional. It transforms the standard reading experience into a "spectacular and amazing" journey through abstract, techie 3D landscapes and tunnels, while prominently showcasing projects and retaining the beloved Easter egg hunt.

## Core Value

To immediately impress visitors with a high-end, fluid, and interactive 3D WebGL experience that highlights Aryan's data/analytics projects and personal brand, completed within a strict 2-day deadline.

### Active

- [ ] Interactive 3D WebGL background scene (abstract, techie, tunnel journey).
- [ ] Scroll-driven camera movement and animations through the 3D scene.
- [ ] Prominent, visually amazing showcase for past projects.
- [ ] Integration of the Easter egg hunt (either 3D or UI-based, TBD).
- [ ] Extremely fast development and implementation timeline (2 days).

### Out of Scope

- A full 10-month custom engine build — Due to the 2-day timeline, we must use established libraries (e.g., Three.js, React Three Fiber, or GSAP) and potentially pre-built shaders/models to achieve the effect quickly.
- Complex backend integrations — The site remains a static/frontend-only application.

## Context

- The current portfolio is a standard HTML/CSS/JS single-page site with smooth scrolling and CSS animations.
- The user feels the current design is not engaging enough and wants a spectacular, immersive user experience similar to awwwards-winning sites (e.g., Jordan Breton's portfolio).
- The timeline is extremely tight (2 days), so the solution must be high-impact but executable quickly.

## Constraints

- **Timeline**: 2 days maximum — The most critical constraint. Scope must be aggressively managed to ensure launch.
- **Tech Stack**: Must use efficient 3D/animation libraries (Three.js, GSAP, etc.) that can be quickly integrated.
- **Performance**: Must remain performant despite the addition of generic WebGL/3D elements.
