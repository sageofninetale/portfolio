# Tech Stack

## Core Technologies
- **Languages:** HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Frameworks:** Vanilla JS (No React/Vue/Angular)
- **Styling:** Vanilla CSS with CSS Variables

## Dependencies & Libraries
- **Icons:** FontAwesome 6.4.0 (via CDN)
- **Fonts:** Google Fonts (Inter)

## Infrastructure
- **Hosting:** Static file hosting assumed (Vercel, GitHub Pages, or Netlify based on typical portfolio)
- **Repo:** Git version control

## Configuration
No build steps or bundlers (e.g., Webpack, Vite) are currently used. The project runs directly in the browser.
