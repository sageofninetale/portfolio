## Task 3: Implement 3D Scene layout and Scroll Bindings
<read_first>
- script.js
</read_first>
<action>
Create a new section in `script.js` titled `// 3D Background Setup` at the top or bottom.
Initialize a `THREE.Scene`, `THREE.PerspectiveCamera`, and `THREE.WebGLRenderer` attached to `document.getElementById('bg-canvas')`.
Add a responsive `window.addEventListener('resize', ...)` handler to update renderer size and camera aspect ratio.
Create an abstract techie environment using points: randomly generate a large number of particles scattered along the Z-axis (forming a tunnel or spatial field) using `THREE.BufferGeometry` and `THREE.PointsMaterial`.
Implement an animation loop with `requestAnimationFrame(animate)` that slowly rotates the entire particle system or moves elements.
Track `window.scrollY` to manipulate the camera's `position.z` or `position.y` within the `animate()` loop or via a scroll event listener, giving a scrolling journey effect.
</action>
<acceptance_criteria>
`script.js` contains `new THREE.Scene()`
`script.js` contains `new THREE.WebGLRenderer`
`script.js` contains a `requestAnimationFrame` loop
`script.js` maps `window.scrollY` (or related offset) to camera/scene transforms
</acceptance_criteria>
